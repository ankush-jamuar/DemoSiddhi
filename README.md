# Siddhidatri Aahar Pvt. Ltd. — Enterprise Website Redesign

A premium enterprise-grade FMCG corporate website redesign concept for **Siddhidatri Aahar Private Limited**, built using **Next.js 15**, **TypeScript**, **Tailwind CSS v4**, **Framer Motion**, and **Shadcn UI**.

This project transforms the company's digital presence from a basic static website into a modern, cinematic, and business-oriented corporate platform focused on:

- Manufacturing excellence
- Corporate trust
- Product branding
- Distributor confidence
- Enterprise-level UI/UX
- Premium FMCG storytelling

---

# 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run production server:

```bash
npm run start
```
